# Workflow : Recommandation de formation

**Acteurs** : Système + RH

## Étapes

1. **Analyse des écarts (skill gap)**
   - Le système analyse les compétences de chaque employé
   - Comparaison avec les compétences requises pour leur poste
   - Identification des écarts de compétences

2. **Détection compétences faibles**
   - Le système identifie les compétences avec un niveau insuffisant
   - Priorisation selon l'importance de la compétence pour le poste
   - Analyse des tendances (compétences en baisse, stagnantes)

3. **Proposition de formations**
   - Le système propose des formations adaptées pour combler les écarts
   - Suggestions basées sur :
     - Le catalogue de formations disponibles
     - Les compétences à développer
     - Le budget formation disponible
     - Les formations suivies précédemment

4. **Validation RH**
   - Le RH examine les recommandations du système
   - Il peut :
     - Valider les formations proposées
     - Modifier les propositions
     - Ajouter des formations supplémentaires
   - Priorisation selon le budget et les besoins stratégiques

5. **Notification employé**
   - L'employé reçoit une notification avec les formations recommandées
   - Il peut consulter les détails de chaque formation
   - Il peut s'inscrire ou exprimer son intérêt

## Inclus

### Historique formations
- Conservation de toutes les recommandations passées
- Suivi des formations acceptées vs refusées
- Analyse de l'efficacité des recommandations

### Analytics budget formation
- Suivi du budget formation par employé, département, compétence
- Prévisions budgétaires basées sur les besoins identifiés
- ROI des formations (amélioration des compétences post-formation)
