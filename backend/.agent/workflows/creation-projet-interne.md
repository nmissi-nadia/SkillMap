# Workflow : Création d'un projet interne

**Acteurs** : Chef de Projet / RH

## Étapes

1. **Création du projet**
   - Le chef de projet crée un nouveau projet dans le système
   - Renseignement des informations de base :
     - Nom du projet
     - Description
     - Client (interne ou externe)
     - Budget
     - Chef de projet responsable

2. **Définition compétences requises**
   - Le chef de projet définit les compétences nécessaires pour le projet
   - Pour chaque compétence :
     - Niveau requis (1-5)
     - Nombre de personnes nécessaires
     - Priorité (critique, importante, souhaitable)

3. **Définition délais & priorité**
   - Date de début et de fin du projet
   - Jalons et livrables clés
   - Priorité du projet (haute, moyenne, basse)
   - Charge de travail estimée (en jours/homme)

## Inclus

### AuditLog
- Enregistrement de la création du projet
- Traçabilité de toutes les modifications du projet
- Historique des compétences requises

### Canal messagerie projet
- Création automatique d'un canal de messagerie dédié au projet
- Tous les membres du projet y ont accès
- Centralisation des communications liées au projet

### Monitoring projets actifs
- Tableau de bord des projets actifs
- Suivi de l'avancement (% complétion)
- Alertes pour les projets en retard ou sous-staffés
- Statistiques sur la charge de travail par projet
